## Como utilizar
```bash
Clone esse repositório:
`git clone https://github.com/zmatezz/portfolio-zmatez/`

Usando terminal, inicie um servidor local:
`npx http-server`

No navegador de sua preferência, digite o seguinte endereço:
`http://localhost:8080`
```
